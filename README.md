# Healthie 🥗

An AI-powered health and nutrition monitoring platform built as a student capstone project.

## What it does
- Upload a food photo → AI identifies the food
- AI estimates calories, carbs, protein, fats, fiber, sugar, and sodium
- Explains benefits and risks in plain language
- Suggests healthier alternatives
- Tracks your full meal history with charts
- Optional medical data tab for deeper health insights

## Tech Stack
- **Frontend**: HTML, CSS (Tailwind), JavaScript, Chart.js
- **Backend**: Python Flask
- **AI**: Google Gemini API (free tier)
- **Database**: SQLite

## Setup Instructions
1. Clone this repository
2. Install Python dependencies: `pip install -r backend/requirements.txt`
3. Add your Gemini API key to `.env`
4. Run the Flask server: `python backend/app.py`
5. Open `frontend/index.html` in your browser
